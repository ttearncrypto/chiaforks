Suggested actions include:
1. Do not keep any funds in any wallets associated with the 24-word mnemonic you are re-using between coins. Instead:
- Create separate "cold" wallets on a different computer, with unique mnemonics for each.
- Transfer any existing balances to these cold wallets.
- Set your farming rewards addresses to these cold wallets.
2. Build the apps from github source code, if possible. Avoid unofficial/unknown Windows installers.
3. Mount your plot drives as read-only to prevent corruption of plot files.
